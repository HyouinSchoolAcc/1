关键洞察 — 这些游戏为什么赚钱？

用户付费的本质不是"游戏性"，而是对虚拟角色的情感投入。抽卡 / 皮肤 / 约会系统的底层逻辑都是"让玩家感受到角色在乎自己"。

年营收上限 ¥10 亿+（头部单款） ¥1–2 亿（头部全产品线） 解锁游戏 NPC"类人对话"付费点
用户画像 18–24 岁 Z 世代男性 18–24 岁 Z 世代 完全重叠 = 同一批用户

💡 我们的不可替代性 — 为什么游戏公司必须找我们？

① 不是"聊天机器人"，是"有记忆的人"：我们的数据训练出的模型记得"上周你说你考试压力大"，今天会主动问"考试怎么样了？"——这种跨日连续性人格是现有所有 AI 伴侣产品做不到的。

② 不是"通用对话"，是"角色专属"：每个角色拥有独立的价值观、经历、能力体系。每个角色都不会说出其他角色的台词，这种一致性来源于七维结构化数据的约束，而非 prompt engineering 的脆弱伪装。

③ 不是"聊天记录"，是"情感时间线"：每条数据包含时间戳 + 亲密度变化 + 角色 / 用户双方日程 → 模型学会了"时间感"和"关系演化"，这让 NPC 能做到"早安问候""记住纪念日""吃醋""冷战后和好"——这些情感交互才是用户付费的终极驱动力。

行业内其他公司（含大厂）仍在收集"一问一答"平面数据。我们独创的 7 维格式（角色日程 / 用户日程 / 价值观 / 经历 / 评判 / 能力 / 带时间戳情感对话）是模型实现"类人持续性人格"的必要条件，且无法通过爬虫或合成数据获取。

6.2 数据飞轮效应

更多数据 → 更好的微调模型 → 更好的 AI 辅助写作工具 → 写手效率提升 → 更多数据 → 成本进一步下降

边际成本递减：随数据量增长，AI 辅助审核将替代部分人工 QC，培训成本由社区"老带新"自然消化。预计产量翻倍时，单条成本可降至 ¥0.05–0.07。
